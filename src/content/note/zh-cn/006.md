# 前言

我们已经在 [Astro 中添加 Giscus 评论系统](005)，那我们也可以写一个留言板界面

# 操作手法

## 1. 新建留言板页面
在 ```pages/[...locale]``` 中新建文件 ```message.astro```

```astro
---
import { getEntry, render } from "astro:content";
import config, { monolocale } from "$config";
import Base from "$layouts/Base.astro";
import i18nit from "$i18n";
import Giscus from "$components/Giscus.svelte";

export async function getStaticPaths() {
  return config.i18n.locales.map(locale => ({
    params: { locale: config.i18n.defaultLocale === locale ? undefined : locale }
  }));
}

const { locale = config.i18n.defaultLocale } = Astro.params;
const t = i18nit(locale);

// 获取留言板内容
const messageContent = await getEntry("information", `${monolocale ? "" : locale}/message`);
const { Content } = messageContent ? await render(messageContent) : { Content: null };
---
```

## 2. 创建 ```message.md```

创建 ```src/content/information/zh-cn/message.md``` 和  ``` src/content/information/en/message.md```  
在其中写下你想说的话  

## 3. 更新翻译

更新翻译文件
```
navigation:
  message: 留言板

message:
  description: 在这里留下你的足迹

navigation:
  message: Message

message:
  description: Leave your message here
```

## 4. 更新导航栏

修改 ```src/layouts/header/Navigator.svelte```，在 Props 中添加 ```message```：

在导航链接中添加
```svelte
<a href={getRelativeLocaleUrl(locale, "/message")} 
   class:location={route.startsWith(getRelativeLocaleUrl(locale, "/message"))}>
	<span>{@render message()}</span>
	<p>{t("navigation.message")}</p>
</a>
```

修改 ```src/layouts/Base.astro``` 中的 ```Navigator``` 组件调用，添加图标  
```astro
<Icon name="lucide:message-square" slot="message" />
```

## 5. End
这样，你的 Blog 就成功加上了留言板。
